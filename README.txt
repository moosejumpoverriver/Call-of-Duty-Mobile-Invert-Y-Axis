

1) Extract Xpadder.rar
2) Start Xpadder.exe as administrator
3) In Xpadder, click controller button on top left
open.., choose: controller_settings_inverted_y_axis.xpaddercontroller
4) In Xpadder,click document button on mid top right
open.., choose: InvertedYAxisCODMOBILE.xpadderprofile
5) Open COD Mobile, set the settings the same as in the image Layout.png. IMPORTANT: Camera (right joystick)
should be set off. 

You can move the mouse around the screen in game to choose options,
select button is the left click to choose menu options
